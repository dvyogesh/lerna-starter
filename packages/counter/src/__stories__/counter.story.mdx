# Counter

## Displays an image that represents a user or organization

Use an Counter for attributing actions or content to specific users. The user's name should _always_ be present when using Counter â€“ either printed beside the Counter or in a tooltip.
<Preview>
    <Story name="standard">
  <Counter>
   {({ increase, decrease, count }) => (
       <div>
         <button onClick={decrease}>-</button>
         <div>{count}</div>
         <button onClick={increase}>+</button>
       </div>
     )}
   </Counter>
</Story>
</Preview>

### Default Values

When no image is supplied to the `src` prop, Counter displays initials. Counter should be used sparingly in situations without access to images.
